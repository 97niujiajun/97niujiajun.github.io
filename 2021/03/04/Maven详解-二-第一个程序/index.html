Maven项目构建命令

在src所在的层级下，在命令行执行命令。

在新项目中，本地仓库还是空文件夹。

首先输入

mvn compile

会开始下载插件。这时本地仓库就有东西了，同时在src同级目录下，多出个target文件夹，里面有classes，和maven-ststus两个文件夹，是程序编译完的东西。

从控制器可以看出，是从阿里云这个私服镜像下载的。

先下插件再编译。

第一次编译会慢，因为要下插件，等以后编译，就很快。

mvn clean

删除target文件夹。

mvn test

会比编译多出两个文件夹：surefire-reports和test-classes文件夹。

并且会在命令行有所显示

mvn package

打包，会在target里面出现一个jar包。

打包时，会自动执行编译、测试，然后才打包。

mvn install

把打包的东西放到本地仓库中去。

gruopId是com.ithema

项目名称是project-java

版本号1.0

用插件创建工程
